## Project setup
```
1) composer install
2) cp .env.example to .env
3) Change DB name, username and password in .env
4) php artisan key:generate
5) php artisan storage:link
6) php artisan migrate
7) npm install
8) npm run dev
9) visit localhost:8000
```
